# 知了租项目盈利分析模拟器

一个基于 Streamlit 的租赁盈利分析工具，用于向合投商户演示"押金首付 + 租金费率"模式下租机业务的现金流、回本周期、底线坏账率等核心指标。

## 项目特点

- **单文件应用**：整个模拟器只有一个 `app.py`（约 800 行），结构简单，便于审计和修改。
- **业务驱动**：参数完全按照实际租机业务设置（押金首付率、平均租金费率、提前还款损失率等）。
- **多维敏感性分析**：内置 4 个敏感性分析图（坏账率、提前还款损失率、押金首付率、平均租金费率），帮助快速判断各参数对盈利的影响。
- **数据导出**：支持把每月现金流明细一键导出为 Excel。

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

依赖清单（参见 `requirements.txt`）：
- `streamlit`：Web UI 框架
- `pandas`：表格数据处理
- `matplotlib`：图表绘制
- `numpy` / `numpy-financial`：数值计算
- `xlsxwriter`：Excel 导出

### 2. 启动应用

```bash
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`。若端口被占用，可指定其他端口：

```bash
streamlit run app.py --server.port 8502
```

### 3. 输入访问密码

应用启动后会要求输入密码（防止未授权访问）。输入正确密码后即可使用模拟器。

### 4. 设置参数并运行

在左侧侧边栏调整参数（详见下方"参数说明"），点击「运行模型」按钮即可：
- 查看每月现金流明细表
- 下载 Excel 报表
- 查看累计净现金流曲线和回本点
- 查看 4 个敏感性分析图

## 参数说明

### 核心业务参数

| 参数 | 范围 | 默认值 | 含义 |
|------|------|--------|------|
| 机器成本 | 1000-15000 元 | 5000 元 | 单台手机的采购成本（也是零售价基准） |
| 每月订单量 | 10-5000 单 | 300 单 | 每月新增的租机订单数 |
| 还款期数 | 6-24 期 | 9 期 | 每个订单的回款总期数 |
| 押金首付率 | 10%-60% | 25% | 客户首月支付的押金首付，按零售价百分比计算 |
| 平均租金费率 | 20%-30% | 25% | 租金价相对零售价的加价费率 |
| 提前还款损失率 | 1%-5% | 3% | 因客户提前结清导致的总租金损失比例 |
| 坏账率 | 0%-10% | 5% | 无法收回的租金占总应收租金的比例 |
| 服务费率 | 0%-10% | 2% | 平台向商户收取的服务费比例 |
| 公司费用率 | 0%-10% | 0% | 公司运营成本占机器成本的比例 |
| 投资比例 | 0%-100% | 100% | 商户实际出资占总出资金额的比例 |
| 投资月份数 | 6-36 月 | 12 月 | 持续新投资金的月份，之后停止投资但仍回款 |

### 核心公式

```
押金首付 = 机器成本 × 押金首付率
总租金   = 机器成本 × (1 + 平均租金费率)
月供     = (总租金 - 押金首付) / 还款期数

# 客户的现金流（对商户来说是回款）
第 1 月：押金首付 + 月供
后续 N-1 月：月供

# 损失调整
提前还款损失后实收总租金 = 总租金 × (1 - 提前还款损失率)
坏账后实收月度回款     = 月度回款 × (1 - 坏账率)
```

## 核心指标说明

运行模型后，主面板会显示以下核心指标：

- **总投资月份**：持续投入新资金的月份数。
- **每月订单量 / 总订单量**：当月和累计的订单规模。
- **最大垫资**：滚动投资下累计净现金流的最小值（绝对值），代表商户需要准备的最高资金水位。
- **累计投资金额**：所有月份投资金额的总和（设备款总投入）。
- **净利润**：模拟周期结束时累计净现金流的终值。
- **实际投资收益率**：净利润 ÷ 累计投资金额。
- **总收益率**：净利润 ÷ 最大垫资。
- **回本周期**：累计净现金流首次由负转正的月份。
- **底线坏账率**：在其他参数固定时，使净利润归零的坏账率水平。商户可据此判断"最多能承受多少坏账"。
- **平均资金投资收益率**：净利润 ÷ 月均垫资，反映"占用资金的单位时间效率"。

## 敏感性分析

应用内置 4 个敏感性分析图，分别考察单一参数变化时对回款总额的影响：

1. **坏账率敏感性**：扫描 0%-8% 的坏账率。
2. **提前还款损失率敏感性**：扫描 0%-8% 的提前还款损失率。
3. **押金首付率敏感性**：扫描 10%-60% 的押金首付率。
4. **平均租金费率敏感性**：扫描 15%-35% 的租金费率。

每个图上的红点表示当前侧边栏参数对应的回款额。

## 常见问题

**Q: matplotlib 图表中文显示成方块？**
A: 确认 `SourceHanSansCN-Regular.ttf` 与 `app.py` 在同一目录，且当前工作目录就是项目根目录。

**Q: 端口 8501 被占用？**
A: 用 `streamlit run app.py --server.port 8502` 指定其他端口。

**Q: 想修改密码？**
A: 编辑 `app.py` 顶部的 `CORRECT_PASSWORD` 常量。注意：密码硬编码在源码中，请勿提交到公开仓库。
